# data-structures-and-algorithms course in UNT 
program 1 : read a CSV file and sort the data using sorting algorithms in C++:
-Insertion Sort
-Merge Sort
-Quick Sort (using first, middle, last median-of-three method for pivot selection)
-Hybrid Quick Sort and Insertion Sort (uses Quicksort for n>20 and Insertion Sort for n<=20)
-Heapsort
program 2 : read a CSV file and write BFS, DFS to show that the graph is connected and then perform Prim's and Kruskal's Algorithm for MSP
